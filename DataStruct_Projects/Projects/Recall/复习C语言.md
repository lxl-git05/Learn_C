## 🧩 功能要求（建议你完整实现）

### ① 联系人结构体设计

```c
typedef struct
{
    char name[20];
    char phone[15];
    char email[30];
    int  age;
} Contact;
```

------

### ② 基础功能（必须）

1. **添加联系人**

   * 边界判断		  (判断输入是否正常)

   - 判断是否已满	（使用宏定义确定最大人数）

2. **显示所有联系人**

   - 指针遍历数组	
   - 格式化输出

3. **按姓名查找**

   - 字符串比较		（重点）
   - 返回索引 or 指针

4. **修改联系人**

   - 指针定位 → 修改成员	（按编号查找or姓名+编号查找）

5. **删除联系人**

   - 数组整体前移			（重要）
   - 非常重要（数据结构前置）

6. **统计功能**

   - 联系人数量
   - 年龄平均值 / 最大值
